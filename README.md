My Metadata
======


Install
-------

Install MyMetadata

    $ pip install -e .

Run
---

    $ export FLASK_APP=MyMetadata
    $ export FLASK_ENV=development
    $ flask run

Or on Windows cmd

    > set FLASK_APP=MyMetadata
    > set FLASK_ENV=development
    > flask run

Open http://127.0.0.1:5000 in a browser.
